## 📖 Overview

**PiMetaConnect** is a groundbreaking application designed to integrate **Metaverse** and **Virtual Reality** technologies with **Digital Payments** powered by the **Pi Network**. This project is a community-driven initiative to support the Pi Network ecosystem while offering a seamless and innovative user experience.

---

## ✨ Features

| Feature                  | Description                                                                 |
|--------------------------|-----------------------------------------------------------------------------|
| **Secure Login**         | Login using Pi Network SDK for enhanced security.                          |
| **Personal Dashboard**   | User-friendly interface for managing user settings.                        |
| **Digital Payments**     | Enable payments using Pi currency seamlessly within the app.               |
| **Multi-language Support** | Interface available in English and Arabic.                               |
| **Security & Privacy**   | Advanced blockchain security to protect user data.                         |

### 🚀 Future Features
- **Instant Messaging**: Support for text, voice, and video chat.
- **File Sharing**: Share multiple file types securely.
- **App Integrations**: Support for Google Drive and Dropbox.
- **Task Management**: Integrated calendar to track events and tasks.

## 🔗 Blockchain Technology

**PiMetaConnect** leverages **blockchain technology** via the Pi Network to enhance security and transparency in digital transactions. Key benefits include:

- **Decentralization**: Data and transactions are stored securely in a decentralized manner.
- **Digital Payments**: Enable seamless and secure payments using Pi currency.
- **Transparency**: All transactions are recorded on the Pi Network blockchain.

---

## 🛠️ System Requirements

- **Node.js**: Version 20 or higher.
- **MongoDB**: For storing user data and transactions.
- **Blockchain Network**: Pi Network (Sandbox for development, Mainnet for production).
- **Vercel CLI**: For deployment to Vercel.
- **GitHub Actions**: For CI/CD setup.

## 📦 Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/Ze0ro99/PiMetaConnect.git
cd PiMetaConnect
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm start
```
